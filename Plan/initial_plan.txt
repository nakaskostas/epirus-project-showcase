-Θέλω να φτιάξω μία web εγαρμογή - static web page που θα κάνει παρουσίαση των έργων που έχει η περιφέρεια Ηπείρου.
-Σκοπός μου είναι να μην γράψω καθόλου κώδικα και όλη τη διαδικασία θα την πραγματοποιήσει ένα llm agent (όπως gemini 2.5 pro).

Σκοπός του web app θα ειναι η παρουσίαση πολυμεσικών αρχείων διάφορων έργων (υπο κατασκευή ή μη) της περιφέρειας Ηπείρου στον ιστό. Θέλω πάντα
ανεξάρτητα από το έργο το web app να κάνει πάντα την ίδια δουλειά που αναφέρω παρακάτω στο ΒΗΜΑ 1 και στο  ΒΗΜΑ 2.


-Το κάθε έργο θα υπάρχει σε έναν φάκελο (μια δομή αρχείων) που θα περιέχει πολυμεσικά αρχεία του έργου. Τα πολυμεσικά αρχεία του έργου κυριως
θα είναι αρχεία εικόνων, βίντεο, κείμενο, pdf με σχέδια εγκαταστάσεων - σχέδια μηχανικών κά. 

ΒΗΜΑ 1
Ο agent θα διαβάζει τον φάκελο που βρίσκονται τα αρχεία του έργου (κυρίως θέλω να δουλεψω με github repositories, δηλαδή όλα τα 
αρχεία του έγου να βρίσκονται σε ένα repository αλλά να μπορεί να διαβάζει και αν ειναι σε ένα τοπικό υπολογιστή ή σε ένα σερβερ όπως
clouflare). Το app θέλω να λειτουργεί ακριβώς το ίδιο ακόμα και αν ο φάκελος ειναι συμπιεσμένος.
Έπειτα θα τροποποιεί τα αρχεία κατάλληλα έτσι ώστε να ειναι έτοιμα για τη παρουσίαση. Δηλαδή ο agent διαβαζει όλα τα πολυμεσικά
αρχεία που αρχικά θα είναι "χύμα" και θα δημιουργεί ένα καινουργιο δομημένο σύστημα αρχείων που θέλω να έχει για κάθε έργο που του αναθέτω
(πχ έναν φάκελο για τα αρχεία εικόνων(και μπορεί και μέσα στον φάκελο υποφακέλους για την κάθε μορφή αρχείων εικόνων), έναν για τα βίντεο κλπ). 
Γενικά να φτιάχνει ένα δομημένο σύστημα αρχείων (φάκελοι και υποφάκελοι) με όλα τα πολυμεσικά αρχεία που θα χρησιμοποιήσει ο agent για το
επόμενο βήμα. 


ΒΗΜΑ 2
Ο agent θα παίρνει τα αρχεία από τον φάκελο που εχει δημιουργήσει στο βημα 1 και θα παρουσιάζει τα αρχεία στον ιστό, προσαρμόζοντας το UI
κατάλληλα με τα επεξεργασμένα αρχεία του έργου που έχει στη διάθεση του. Δηλαδή για παράδειγμα θα προσαρμόζει αρχεία κειμένου (όπως excel
ή word αρχεία αν υπάρχουν) και θα εμφανίζει το κείμενο δίπλα από κάποια εικόνα ή βίντεο πχ έτσι ώστε ολόκληρο το website να έχει μια 
μοντέρνα δομή και εμφάνιση. 

-Θέλω όλη την δουλειά να την κάνει ο agent, δηλαδή μέσω CLI να έχει πρόσβαση ο agent στο σύστημα αρχείων, στον browser και γενικά παντού 
ώστε να τα κάνει όλα ο agent.

-Το website/webapp της παρουσίασης των έργων θέλω να το δημιουργήσει ο agent.

--UI
Θέλω μία στατική ιστοσελίδα με μοντέρνο σχεδιασμό. Το ui να δίνει εμφαση στην παρουσίάση των πολυμεσικών αρχείων κατά κύριο λόγο.



-Το webapp θέλω να λειτουργεί ανεξάρτητα από το έργο που του έχει ανατεθεί, δηλαδή οποιοδήποτε έργο και να του δώσω
να μπορεί να κάνει την online παρουσίαση για το συγκεκριμμένο έργο.

-Το περιεχόμενο του website θα είναι στατικό.

-Όλα θα είναι σε μία σελίδα και το μενού του website θα λειτουργεί σαν anchors μέσα στη σελίδα.
